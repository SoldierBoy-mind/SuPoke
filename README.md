## ⚙️ Requirements

Make sure you have installed:

* Node.js (LTS version recommended)
* npm

---

## 📦 Installation

Clone the repository from GitHub:

```bash
git clone https://github.com/SoldierBoy-mind/SuPoke.git
cd SuPoke
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run the app

Start the server:

```bash
node server.js
```

Or (if you use npm scripts):

```bash
npm start
```

---

## 🌐 Usage

Open your browser and go to:

```
http://localhost:3000
```

(Change the port if needed)
